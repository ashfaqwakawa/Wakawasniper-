# Wakawa Sniper Bot

A Solana Telegram sniper bot with:
- 30-user limit
- Unique internal wallet for each user
- Deposit / Balance system
- Helius RPC support
- Jupiter real swap engine (buy & sell)
- Rug / honeypot / safety protection
- Trending scanner
- Whale alerts
- Fast 60-second price engine
- Fully Railway deployable

## Files
bot.py  
requirements.txt  
Procfile  
README.md  

## Env Variables (Railway)
BOT_TOKEN  
HELIUS_RPC  
FERNET_KEY  
ADMIN_ID

## Deployment Steps
1. Upload all files to GitHub
2. Connect repo to Railway
3. Add environment variables
4. Deploy
5. Bot is live
